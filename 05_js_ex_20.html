Attribute 제어하기
    1. 클래스 추가/삭제
      - 추가 : addClass() 함수
        http://api.jquery.com/addClass/
      - 삭제 : removeClass() 함수
        http://api.jquery.com/removeClass/

    2. HTML 속성 제어
      - 추가/변경 : attr() 함수
        http://api.jquery.com/attr/
        예) $('box1').css('width','300'); class가 box1인 요소의 가로길이를 300px로 지정해라
        예) $('box1').css('width'); class가 box1인 요소의 가로기리이를 읽어 들여라

      - 삭제 : removeAttr() 함수
        http://api.jquery.com/removeAttr/

      ** prop() : HTML 속성 중 정보가 아닌 상태를 다루는 속성 제어할때 사용한다.
      http://api.jquery.com/prop/
      예) input tag중 readonly, checked, disabled
